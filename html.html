    Q: HTML语义化是什么意思参考答案

    A: 
    HTML语义化指赋予HTML标签一定的语法结构意义，这种意义除了在开发时有一定的意义，避免一些标签的滥用如一些早期的网站中使用div+span布局开发整个网站，还给搜索引擎提供帮助更好地理解网站的内容

    Q:  讲述HTML、CSS、JS分别在网页中扮演怎样的角色 参考答案

    A: 
    HTML相当于网页的内容与结构
CSS则负责网页的外观和样式
JS负责网页的功能交互与行为

    Q:  谈谈html中一些常用的语义化标签 参考答案

    A: 
    header 头部
footer 尾部
nav：导航

main 呈现文档的主体部分，独一无二的内容。
section 表示文档中一个通用独立章节，通常含一个标题
article：表示一个文章区域。里面可包含h1-h6标题，article可互相嵌套。

aside：表示和页面其他内容无关的独立部分。侧边栏或抽屉
address 联系信息

    Q:  <head>内有哪些有用的标签？ 参考答案

    A: 
    一部分是元数据，包括设置网页的字符集charset、标题title、网页的作者author、描述description、声明网站相关的关键词keywords，还可设置其他类型的元数据，需遵循特定社交网站"元数据协议"。
如设置应用于facebook：
<meta
  property="og:image"
  content="https://developer.mozilla.org/mdn-social-share.png" />
<meta
  property="og:description"
  content="The Mozilla Developer Network (MDN) provides
information about Open Web technologies including HTML, CSS, and APIs for both Web sites
and HTML Apps." />
<meta property="og:title" content="Mozilla Developer Network" />

除此之外包括网页的CSS样式等

    Q:  介绍一下<button>元素 参考答案

    A: 
    <button>表示一个可点击的按钮，可通过disabled设置其可交互性，搭配form进行表单提交或处理非常方便
如通过form属性（为一个表单的id）链接表单（可以不是button的祖先元素）从而覆盖表单的一些属性：
formaction，formenctype，formmethod，formtarget，name，value（name和value会随表单一同提交到action指定的地址）

另type属性（submit,button,reset）默认为submit，如想监听、处理按钮事件可设置为button

    Q:  介绍一下HTML中一些常见有用的全局属性 参考答案

    A: 
    经常使用的：
id
class
style
title:元素相关信息的文本
lang:定义元素的语言
data-*：自定义数据属性
tabindex:指定元素是否可聚焦（负值不可导航聚焦，可通过js脚本手动聚焦），以及它参与键盘导航（tab键）的顺序。顺序为：正值中的小值 > 正值中的大值 > 隐式值(没有显示指定如表单、链接等可视为0)
draggable:指示是否可以使用 Drag and Drop API 拖动元素
hidden：用来隐藏页面中的元素
translate：指定在页面本地化时是否转换元素的属性值及其 Text节点子节点的值

不常使用或使用频率低，但有特殊场景的：
contenteditable：元素是否可被用户编辑
autofocus：这个东西好用用于设置自动聚焦（前提是元素自带聚焦特性如表单，或者设置了恰当tabindex属性的普通元素）
inert：使浏览器忽略元素上用户的输入事件（点击、聚焦、输入等）
role：定义内容的语义，ARIA无障碍相关。
dir：指示元素中文本方向。有左到右和右到左。

有些鸡肋的：
accesskey：设置快捷键来聚焦对应的属性所在的元素。兼容性、实用性差不建议用
autocapitalize：控制用户的输入如何自动大写，在手机上搭配软键盘使用。兼容性差不建议用
virtualkeyboardpolicy：用于控制屏幕上的虚拟键盘行为，需搭配contenteditable使用。兼容性不好

    Q:  <input>元素常见的使用场景 参考答案

    A: 
    最常见：
输入框text、单选框radio、密码password、复选框checkbox、文件选择框file、普通按钮button、重置按钮reset、提交按钮submit

除此之外还有一些有用但不常用的：
颜色拾取器 type=color（value为16进制的HEX值）
日期选择器 type=date (value的格式为 yyyy-mm-dd) ，可通过min和max属性限定日期选择的区间
日期+时间选择器 type=datetime-local （value'格式为yyyy-MM-ddThh:mm ）
月份选择器 type=month value格式为yyyy-MM
时间选择器 type=time value格式为hh:mm（24小时制）
星期选择器 type=week value格式为：2017-W01

还有一些其他有用的type如：
email、search、tel、url等有更好的语义性。

image用于创建图形化按钮。number用于限制输入为数字、滑块range

    Q:  谈谈<base>元素 参考答案

    A: 
    指定文档中所有相对URL的根 URL。不仅作用于页面内的链接，也包括发出的ajax请求

    Q:  如何创建可点击的图像区域 参考答案

    A: 
    一种方法是在<img>上添加区域覆盖层，对覆盖层增加事件监听，在处理函数里处理点击逻辑
另一种通过<map>+<area> ，与<img>联结后，去设置区域的形状（圆形、矩形、多边形等）和位置，以及链接的地址URL即可。

    Q:  <datalist>元素有什么用 参考答案

    A: 
    包含了一组<option> 元素，给其他表单元素提供可选值，可用于根据输入进行联想功能。不做做静态的联想倒挺方便，动态联想就算了吧，鸡肋！
